## 1. Background and Overview

The Diwali season is one of the largest retail events in India, generating significant sales across multiple categories. To understand customer purchasing behavior and help businesses improve their festive-season strategy, this project analyzes a real-world Diwali Sales dataset.

The dataset contains 11,253 rows and 15 columns, including customer demographics, purchase details, product categories, and order amounts. Using Python-based data analysis, this project aims to uncover insights that can help companies improve customer experience, target the right audience, and increase revenue during festive campaigns.

This analysis was performed in Jupyter Notebook using tools such as Pandas, NumPy, Matplotlib, and Seaborn.
